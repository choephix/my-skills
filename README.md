# my-skills

Personal Zo skills.

## Included

- `generate-with-fal` — fal.ai generation skill for image, image-editing, and 3D workflows.

## Layout

Each skill lives in its own directory and is self-contained:

- `SKILL.md` defines how to use the skill
- `scripts/` contains the runnable tools
- `assets/` stores registries or static data
- `references/` holds supporting docs
